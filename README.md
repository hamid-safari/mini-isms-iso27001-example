# Mini ISMS Example – ISO 27001

Dieses Repository enthält ein vollständiges, beispielhaftes Information Security Management System (ISMS) auf Basis von ISO/IEC 27001. 
Es dient als Demonstration der grundlegenden Strukturen, Dokumentationen und Methoden, die für ein ISMS notwendig sind.

## Inhalte
- ISMS Scope
- Asset Register
- Risk Assessment (inkl. Risikomatrix)
- Annex A Mapping
- Security Policies
- Incident Response Playbooks

Alle Inhalte sind rein exemplarisch und nicht für produktive Umgebungen gedacht.
