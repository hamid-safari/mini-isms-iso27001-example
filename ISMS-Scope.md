# ISMS Scope Statement

## 1. Purpose
Der Scope definiert den organisatorischen und technischen Geltungsbereich des Beispiel-ISMS.

## 2. Scope Definition
Das ISMS umfasst:

- Digitale Informationsverarbeitungssysteme
- Interne IT-Infrastruktur (Netzwerk, Server, Endgeräte)
- Kritische Unternehmensprozesse: Kundenverwaltung, Abrechnung, interne Kommunikation
- Personal mit Zugriff auf sensible Daten
- Cloud-basierte Systeme (SaaS)

## 3. Ausschlüsse
- Private Geräte (BYOD)
- Nicht-IT-bezogene Geschäftsbereiche

## 4. Verantwortlichkeiten
- Information Security Officer (ISO): Verantwortlich für Aufbau und Pflege des ISMS
- Management: Freigabe und Unterstützung
